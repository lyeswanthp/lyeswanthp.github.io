Intelligent Movie Recommendation System

View on GitHub →

Overview

A dual-mode movie discovery application that employs intelligent search mechanisms to help users find films matching their preferences. Users can either describe desired movies in natural language or request suggestions similar to films they already enjoy. Built with both lightweight and advanced versions to suit different deployment scenarios.

Key Features

  • Dual Search Modes: Description-based natural language search and title-based similarity matching
  • Basic Version: Fast TF-IDF search working completely offline without internet dependencies
  • Advanced Version: Semantic search using ChromaDB for understanding meaning beyond keywords
  • Optional AI Explanations: Ollama integration for generating contextual recommendations
  • Graceful Degradation: Advanced features work independently without external AI dependencies
  • Configurable Results: Choose between 3, 5, or 10 recommendations per query

Technical Approach

The system demonstrates thoughtful engineering by offering both lightweight and feature-rich implementations. The basic version uses TF-IDF for quick keyword matching, perfect for offline scenarios and resource-constrained environments. The advanced version incorporates semantic search with ChromaDB, allowing it to understand the meaning and context of queries rather than just matching keywords.

The optional Ollama integration adds AI-generated explanations for why movies are recommended, but the system is designed with fallback functionality—all core features work without Ollama, showcasing robust design principles.

Technologies Used

Python Flask ChromaDB TF-IDF Ollama Semantic Search NLP MovieLens Dataset

Dataset

Utilizes the MovieLens metadata containing thousands of films with comprehensive descriptions, genres, and metadata. The rich dataset enables both accurate keyword matching and semantic understanding of movie characteristics.

Impact

This project showcases practical AI engineering by providing multiple deployment options—from a lightweight offline version to a sophisticated semantic search system with optional AI enhancements. The architecture emphasizes accessibility and robust design, making it suitable for various use cases from personal projects to scalable web applications.
